# 📊 Customer Segmentation & Sales Analysis (Power BI)

## 🔍 Project Overview
This project focuses on analyzing customer behavior and sales data using Power BI.  
RFM (Recency, Frequency, Monetary) analysis is applied to segment customers and derive business insights.

---

## 🚀 Key Features
- Data cleaning using Power Query  
- Customer segmentation using RFM analysis  
- Interactive dashboard with KPIs  
- Revenue trend analysis  
- Country-wise sales performance  
- Top N filtering for insights

## 🧠 Key Insights
- Champions contribute the highest revenue  
- Many customers fall under the "At Risk" category  
- United Kingdom generates the majority of sales  
- Revenue shows a declining trend
